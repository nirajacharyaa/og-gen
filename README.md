# OG-gen (Open Graph Image generator)

made with @vercel/og(satori)

# Usage

### Installation
```bash
# clone the repo

git clone link

# install dependencies

pnpm i

# make changes

# run the server

pnpm dev


```

### API

```txt
/*
*   title and date are required.
*/

https://localhost:3000/api/og?title=someTitle&date=someDate

```
